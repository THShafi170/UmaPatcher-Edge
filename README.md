UmaPatcher is an UM:PD Hachimi mod patching tool for Android. Its main purpose is to provide you with translation patches within the game, among other features.

For more information, please check out [Hachimi Edge](https://github.com/kairusds/Hachimi-Edge).

# Requirements
- Android 7.0 or above

# Download
Download the latest release from the [Releases page](https://github.com/THShafi170/UmaPatcher/releases).
